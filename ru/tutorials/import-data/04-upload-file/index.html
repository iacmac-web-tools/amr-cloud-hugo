Шаг 1 - Загрузка файла

Загрузка

После ознакомления с разделом Важная информация пользователь может приступить к загрузке данных в AMRcloud. Для этого необходимо перейти на вкладку “Шаг 1”, нажать на кнопку “Browse…" и выбрать файл с данными для загрузки. После успешной загрузки таблицы необходимо нажать кнопку “Проверить и сохранить шаг 1”.

Важно!

  • AMRcloud поддерживает загрузку таблиц в формате .xlsx и .csv.
  • Предварительный просмотр таблицы настраивается пользователем на отображение от 5 до 30 строк с помощью специального ползунка.
  • В панеле “Настройки для csv файла” настраивается отображение таблиц, загруженных в формате .csv (в таких таблицах отсутствуют столбцы, и вся информация представлена в виде строк).

Как сохранить файл в csv

Для того, чтобы сохранить таблицу в формате .csv необходимо в Microsoft Excel перейти во вкладку Файл (левый верхний угол), выбрать пункт Экспорт → Изменить тип файла → Другие типы файлов → CSV (разделители запятые)(*.csv).

Проблемы с кодировкой текста

Пользователь может столкнуться с ситуацией, когда в таблице появляются непонятные знаки (т.н. “иероглифы”) вместо текста. Разберем решение данной проблемы на конкретных примерах.

Пример 1

При загрузке таблицы в AMRcloud на Шаге 1, в окне предварительного просмотра появляется надпись “Не могу прочитать файл”

Алгоритм решения:

  1. Скачать и установить бесплатный текстовый редактор Sublime Text (использование текстового редактора Sublime Text носит рекомендательный характер и не является оплаченной формой неличной передачи информации; пользователь может использовать любой имеющийся текстовый редактор с настройкой кодинга).

  2. В установленном редакторе выбрать раздел file → Open File.

    Выбрать в появившемся окне таблицу, которую не удалось открыть в AMRcloud. Открывшийся документ будет представлен в виде сплошного текста.

  3. Поочередно изменять различные кодировки и ориентироваться на текст (“иероглифы” должны исчезнуть). Рекомендуется начинать с UTF-8 и Windows-1251. Раздел File → Reopen with Encoding → необходимая кодировка.

    После выполненных действий пользователь сможет прочитать текст. “Иероглифы” исчезнут.

  4. Файл необходимо сохранить. File → Save with Encoding → UTF-8.

Пример 2

При загрузке данных в окне предварительного просмотра на Шаге 1, проблем не было. Пользователь провел анализ данных, скачал таблицу из AMRcloud, но Microsoft Excel открывает данные с “иероглифами”.

Алгоритм решения №1

Такая ситуация могла произойти, если пользователь изменил кодировку при сохранении файла. Необходимо ещё раз скачать таблицу с кодировкой по умолчанию, т.е. Windows (кириллица).

Алгоритм решения №2

  1. Скачать и установить бесплатный текстовый редактор Sublime Text (использование текстового редактора Sublime Text носит рекомендательный характер и не является оплаченной формой неличной передачи информации; пользователь может использовать любой имеющийся текстовый редактор с настройкой кодинга).
  2. В установленном редакторе выбираем раздел File → Open File.
  3. Файл необходимо сохранить с кодировкой Windows-1251. File**→** Save with Encoding **→** Windows-1251.

Теперь можно просматривать таблицу, которая была скачана из онлайн-платформы AMRcloud, на локальном компьютере.
