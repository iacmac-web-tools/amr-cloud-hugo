AMRcloud January update

Posted on 30 January 2023

AMRcloud has been updated to version 0.9.2:

  • added ability to add charts from “Data Structure” tab (subtab “Plot by”);

  • added ability to add charts from “Selected Antibiotic” tab (subtab “Plot by”);

  • added ability to disable numerical captions on the charts.

  • added ability to change the plot display settings (size and visibility of captions) in the dashboards
