Step 3 - Geocode

Geocoding is an important step in creating a basis with which AMRcloud builds various maps and pins geographic markers.

Regions, cities and even separate hospital departments can be used as geographical objects in AMRcloud.

General Information

In order to start the geocoding process you should press the «1.Geocode it» button. This process takes some time, so user should wait for the system to put coordinates in the table below and for markers to appear in the map preview window.

In order to open the map preview, user should press the header of the «Map: preview» blue panel.

After the user is sure that all markers with coordinates are placed correctly, he should press the «Validate Step Three» button and proceed to the next import stage.

If Location(s) Is Not Found

If AMRcloud didn’t recognize the geographical object (didn’t put coordinates next to its name), user should check the correctness of the name of the geographical object. If there are errors in the name(s), it is necessary to correct the data in the source data table, to reset all import steps, and to upload the newly corrected version.

If there are no errors in the names of geographical objects, user should manually set the coordinates. To do it he should enter the corresponding coordinates in the right table (orange panel «Location(s) not found») and press the button «Fix Coordinates».

How to Get the Coordinates

User can get the coordinates of the required geographical objects using the preview map.

. To do it, user should zoom in on a geographical object and click on it with the left mouse button. The object’s Latitude and Longtitude will be displayed below the map. After determining the object’s coordinates, user should copy and paste them into the corresponding fields.. If user doesn’t know the exact location of the geographical object on the map, he can use search string on the map and then left-click the object..

Incorrect Marker’s Position

The AMRcloud system may not automatically recognize user’s geographical objects. For example, instead of St. Petersburg, Russia, system may pin St. Petersburg, Florida, USA. Therefore, user should carefully check the preview map before saving the results of the geocoding stage.

If user wants to correct the position of the marker on the map, he should:

  1. Click with the left mouse button on the required cell in the green table.
  2. Press the «Edit Coordinates» button.
  3. Edit the coordinates in the orange table.
  4. Press the «Fix Coordinates» button.

Local Objects

If user often works with his specific set of local geographical objects, he can save the coordinates of these local objects for future automatic use. For this purpose after editing coordinates in orange table, user should check the box «Save coordinates for future usage». So the geographical coordinates will be permanently assigned to the corresponding name of the geographical object and will be saved in the user account. At subsequent downloads, the coordinates of objects with the same names will be automatically geocoded.
