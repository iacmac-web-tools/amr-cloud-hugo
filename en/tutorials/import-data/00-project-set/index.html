Projects and Data Sets

What is a Project / Data Set

Within the AMRcloud user can create different projects and data sets.

Project is a structural element of the AMRcloud platform that allows to group several data sets at the user’s discretion.

Data set(s) is a basic unit within the project that contains information with which AMRcloud works directly.

How to create a project

In order to create a project, user should press the + New Project button in the upper left corner of the screen after entering the personal account.

The modal dialog «Add new project» invites user to fill in the «New project name» and «Project subtitle» fields (with a limit of 255 characters). It is strongly recommended to use unique or significantly different names for your projects. Do not limit yourself to the «test1» and «test2» variants. The project subtitle can be represented by keywords that explain the name.

Types of projects

User can work with two types of projects: - My Project - Team Project

My Project is a personal user project. Access to the data is granted to the owner alone. The owner can share his project’s data with other users by generating external public links.

Team Project is a project for working with other users in a team. One of the users becomes a team administrator (i.e. creator), and others become team members (i.e. users who invited by the team administrator).

Data sets

Within the project user creates several data sets. To create a new data set, the user needs to open the project by clicking the «Open» button and then click the »+ New Set» button.

The modal dialog «Add New Set» invites user to fill in the «New set name» and «Set subtitle» fields (with a limit of 255 characters). It is strongly recommended to use unique or significantly different names for your sets. Do not limit yourself to the «test1» and «test2» variants. The set subtitle can be represented by keywords that explain the name.

After creating the project and the data set, user can start uploading the data. To do it, the user needs to open the project, to click the left mouse button on the data set and then to click the «Edit Set» button.
