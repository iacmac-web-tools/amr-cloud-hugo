AMRcloud. Импорт данных

Руководство «AMRcloud. Импорт данных» предназначено для ознакомления с основными этапами загрузки пользовательских таблиц в онлайн-платформу AMRcloud , индивидуальной настройкой рабочей среды и подготовкой системы к непосредственному анализу и визуализации данных.

Все шаги импорта данных детально описаны с включением практических примеров.
