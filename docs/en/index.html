AMRCloud

AntiMicrobial Resistance Cloud is a web platform for analysis and sharing of AMR surveillance data.

Log in

Now AMRCloud works in test mode and access is granted by invite.

The AMRcloud platform allows users to

create personal accounts with secure access

manage multiple AMR projects and datasets

import metadata and AST results directly from Excel or CSV files using a wizard

(minimum inhibitory concentrations (MICs), inhibition zone diameters, susceptibility categories (S/I/R), resistance markers)

automatically determine clinical S/I/R categories for the known organism-antibiotic combinations

from MIC or inhibition zone diameter data using the European Committee on Antimicrobial Susceptibility Testing (EUCAST) breakpoints

explore data using smart filters and interactive visualization tools

(tables, charts, maps)

export statistics and infographics

generate web links and share results

(selected infographics or entire web projects)
